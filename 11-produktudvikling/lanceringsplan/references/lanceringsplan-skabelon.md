# Lanceringsplan - skabelon og miniature-eksempel

Planen skal kunne læses af en medarbejder der ikke var med til møderne, og eksekveres uge for uge. Hver aktivitet har ejer og dato - "vi bør også..." uden navn og dato er en idé, ikke en plan.

## Skabelonen

```
# Lanceringsplan: [navn] - lancering [dato]

## Konklusion
[3 linjer: hvem vindes først, gennem hvilken hovedkanal, og det vigtigste
måltal med deadline]

## Brohoved-segmentet
| Kandidat | Brændende problem | Betalingsvilje | Kan nås | Henvisningskraft |
|---|---|---|---|---|
| [segment A] | høj/mellem/lav + belæg | ... | ... | ... |

Valgt: [segment] fordi [én linje]. Næste segmenter efter brohovedet: [hvem].

## Budskabet
Kernen (fra vaerditilbud): "[sætningen]"
| Målgruppe | Budskabslinje | Belæg (med kilde) |
|---|---|---|

## Kanalerne (max 3)
| Kanal | Hvorfor | Vigtigste aktivitet | Grov pris |
|---|---|---|---|
Fravalgt: [kanal] fordi [én linje].

## Tidslinjen
### Før lanceringen (uge -4 til -1)
| Aktivitet | Ejer | Dato |
|---|---|---|
| [fx forvarsel til nyhedsbrevsliste + venteliste] | | |
| [fx materiale bestilt hos linkedin-opslag/nyhedsbrev - som kladder] | | |
| [fx briefing af medarbejdere: hvad svarer vi kunder] | | |

### Lanceringsugen
| Aktivitet | Ejer | Dato |
|---|---|---|

### Efter (dag 8-90)
| Aktivitet | Ejer | Dato |
|---|---|---|
| [fx opfølgning på ikke-åbnere / genopslag med ny vinkel] | | |
| [fx kundecase fra første kunde (kundecase-skillen)] | | |
| [fx dag 30- og dag 90-evaluering mod succeskriterierne] | | |

## Succeskriterier
| Måltal | Deadline | Over grænsen → | Under grænsen → |
|---|---|---|---|

## Risici og mitigering
| Risiko | Sandsynlighed | Mitigering |
|---|---|---|

## Budget
Ramme: [kr. + timer]. Fordeling pr. kanal: [...]. Tilvalg uden for rammen
markeret [TILVALG].

## Antagelser og åbne spørgsmål
[altid med - også når tom: skriv "ingen"]
```

## Miniature-eksempel (uddrag - fiktiv bogholderydelse)

```
## Konklusion
Vi lancerer bilagsrykker-abonnementet til de 430 bogholdere på egen
nyhedsbrevsliste, med opsøgende opfølgning på de 40 varmeste. Vigtigste
måltal: 5 betalende inden dag 30.

## Succeskriterier
| Måltal | Deadline | Over → | Under → |
|---|---|---|---|
| 25 på venteliste | dag 0 (før lancering) | lancér | udskyd + test budskab |
| 5 betalende | dag 30 | åbn kanal 3 (LinkedIn-indhold) | interview de nej-sigere |
| 15 betalende | dag 90 | skalér annoncer | revidér pris/pakke |
```

Bemærk formen: hvert tal har en deadline OG en konsekvens i begge retninger. Det er dét, der gør planen til en plan.
